# 📋 Lista de Nomes

Uma aplicação web simples e funcional para gerenciamento de nomes, desenvolvida para praticar manipulação do DOM e armazenamento local de dados no front-end.

🚀 **[Clique aqui para ver o projeto rodando na prática](https://gabrielfresta.github.io/listas-de-nomes/)**

---

## 🛠️ Tecnologias Utilizadas

* *HTML5*: Estruturação semântica da página.
* *CSS3*: Estilização e layout responsivo.
* *JavaScript (Vanilla)*: Manipulação de eventos, lógica de adicionar/remover itens e interação com o usuário.
* *localStorage*: Persistência de dados para manter a lista salva mesmo após fechar o navegador.

---

## 📌 Funcionalidades

* Adicionar novos nomes à lista.
* Remover nomes existentes.
* Persistência de dados simples utilizando localStorage.
* Interface simples e intuitiva.
